The language is primarily used within web browsers, and has the ability to access built-in objects in other applications. It is also used in server programming, such as in runtime environments such as Node.js. JavaScript was originally developed by Brendan Eich of Netscape Communications Corporation, first under the name Mocha, later as LiveScript, and eventually as JavaScript.
